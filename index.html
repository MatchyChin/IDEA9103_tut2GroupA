IDEA9103 Design Programming Creative Coding Major Project

In this project, we have selected the artwork from Piet Mondrian 'Broadway Boogie Woogie'

We first recreate the artwork in p5.js coding with a touch of responsive change which the size of the artwork will change based on the window size. We then develop our creation in different approaches such as Audio, Time-Based, and User Input to create different animations.

I have modernized Piet Mondrian's iconic "Broadway Boogie Woogie" by opting for vibrant macaron hues in place of the primary colors traditionally used.

The shift to these brighter, more vivid tones aims to infuse the artwork with a youthful and contemporary spirit.

Through user interaction with the mouse, a novel visual experience unfolds, ingeniously blending classic art with current trends.
